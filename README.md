# NovaStream Seite mit Nutzer-System

Enthalten:
- Landingpage
- Registrierung
- Login
- Dashboard
- SQLite Datenbank
- Admin-Ansicht aller Nutzer

## Demo-Zugänge
- Admin: admin@novastream.local / Admin123!
- Test: test@novastream.local / Test123!

## Start
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Dann öffnen:
http://127.0.0.1:5000
